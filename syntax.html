KivaDoc Syntax

Document Structure

Here is an example of a simple document: 

Document title
====================
toc:auto
attribute2:foo

Document body...

The first line of the document is the document title. The second line is a divisor-line (one or more "="-characters). Then comes a list of attributes; key:value, key:value.

After an empty line, the document itself begins.

Document Attributes

Currently recognized attributes are:

Attribute

Description

toc:auto

Automatically generate a table-of-contents for the document.

style:<style-sheet-path>

Includes the CSS style sheet to the document.
